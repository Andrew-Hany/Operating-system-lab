how to backup tool:

1) the backup.sh is inside the 01 folder
so when i run it i write "./01/backup.sh 01 backup 123"
2)so in order to run the backup tool, you need to do the following 
    a) ./path/to/backup.sh #firstfolder #secondfolder #password
    b)first folder -> is the folder you want to backup
    c)second folder -> where you want to backup
    d)password -> the password you will open with the backup folders 

    Note: exclude "/root/" from the path of the folders above
